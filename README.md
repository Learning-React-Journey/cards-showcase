# Cards Showcase

A React project that displays hotel cards with image, title, description, rating, and price.

## Concepts Learned and Applied
- **Components**: Built reusable Card component.  
- **Props**: Passed data like image, title, rate, description, and price to components.  
- **Data Mapping**: Used array map function to render multiple cards dynamically.   
- **External Libraries**: Used FontAwesome for icons.
